# STIDFA

**Switch to Pay**

STIDFA is a cross-border payment platform designed to enable seamless money movement between Nigeria (NGN) and Francophone West Africa (CFA).

Our goal is to make sending, receiving, and using money across borders as simple and reliable as local transfers.

---

## 🚀 Overview

Cross-border payments in the Nigeria ↔ Francophone Africa corridor are still largely dependent on:

- Cash-based systems  
- Agent networks  
- Informal transfer methods  
- Delayed or fragmented digital solutions  

STIDFA is building a **cashless-first infrastructure** that removes these limitations.

---

## 🎯 What We Are Building

A mobile-first system that allows users to:

- Send money across borders (NGN ↔ CFA)  
- Receive funds instantly  
- Use funds digitally without cash withdrawal  
- Experience automatic currency conversion  

---

## ⚙️ Core System Components

The platform is built around:

- **Mobile App (Flutter)**  
  User interface for sending, receiving, and managing funds  

- **Backend API Layer**  
  Handles requests, routing, and partner integrations  

- **Ledger System**  
  Double-entry accounting system for accurate balance tracking  

- **Transaction Engine**  
  Processes transfers with idempotency and retry logic  

- **FX Engine**  
  Handles NGN ↔ CFA conversion logic  

- **Partner Integration Layer**  
  Connects to local financial systems in Nigeria and Francophone markets  

---

## 🔐 Key Principles

- **Reliability first** — transactions must not fail silently  
- **Cashless experience** — minimize dependency on physical agents  
- **Clarity over complexity** — simple user flows  
- **Trust-driven design** — transparency in every transaction  

---

## 🛠️ Tech Stack (MVP)

- **Frontend:** Flutter  
- **Backend:** Node.js (NestJS) or Python (FastAPI)  
- **Database:** PostgreSQL  
- **Queue System:** Redis  
- **Infrastructure:** AWS / Render
